# Exercise Demo Files

Place your exercise demo files here.

Supported formats:
- `.mp4` / `.webm` (video, preferred)
- `.gif` (animated image)
- `.json` (Lottie animation)

After adding a file, register it in `src/data/demoRegistry.ts`.

Example:
```ts
'Goblet Squat': { type: 'video', src: '/demos/goblet-squat.mp4' },
```
